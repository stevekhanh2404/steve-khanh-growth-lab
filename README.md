# Steve Khanh Growth Lab

Website tĩnh gồm 6 công cụ không cần AI API:

1. Digital Growth Diagnostic
2. Content Strategy & Calendar Planner
3. Media Plan & Budget Simulator
4. KPI Tree Builder
5. E-commerce Profitability Tool
6. Campaign Naming & UTM Builder

## GitHub Pages

1. Tạo repository `steve-khanh-growth-lab`
2. Upload `index.html` và `README.md`
3. Settings → Pages
4. Deploy from branch → `main` → `/ (root)`

Địa chỉ dự kiến:
`https://stevekhanh2404.github.io/steve-khanh-growth-lab/`

## Ghi chú
- Không cần API key.
- Không cần backend.
- Logic chạy trực tiếp trên trình duyệt.
- Chart.js được tải qua CDN.
